# About OpenNest

**OpenNest** is a **technical builder community at Lovely Professional University** designed to bring together students who want to **learn deeply, collaborate effectively, and build meaningful projects**.

OpenNest is built using research from **India's top technical clubs** including IITs, NITs, BITS, and IIITs — combining **best practices, structure, and long-term sustainability**.

OpenNest is not an event club.
OpenNest is a **builder ecosystem**.

---

# Vision & Mission

## Vision

> To create the most serious and collaborative technical builder community at LPU — where students learn, build, and ship impactful solutions.

## Mission

> Build an ecosystem where students grow through collaboration, real-world projects, mentorship, and documentation-driven learning.

---

# The OpenNest Model

OpenNest operates in **three layers**:

```
Open Community → Domain Pods → Core Council
```

### Open Community

* Open to all students
* No prerequisites
* Learning sessions & small tasks

### Domain Pods

* Selected contributors
* Work on real projects
* Mentorship-based learning

Domains:

* Product & Systems
* AI/ML & Data
* Security & Reliability

### Core Council

* 3–5 core members
* Club leadership
* Strategic direction

This structure ensures **quality + scale simultaneously**. 

---

# Core Principles

OpenNest is guided by five core principles:

### Build Over Talk

Output matters more than titles.

### Small Serious Core

Lean inner team, open outer community.

### Public by Default

Open repositories and documentation.

### Mentor What You Master

Knowledge flows downward.

### Respect Time and Rigor

Meetings and work both matter.

These principles are inspired by research of top Indian technical clubs.

# Project Philosophy

OpenNest follows a **3-phase project pipeline**

### Phase 1 — Foundation

Learning, skill-building, documentation

### Phase 2 — Early Builds

Small, practical projects

### Phase 3 — Flagship Projects

Large, long-term impactful builds

Projects evolve naturally as members grow. 

---

# Mentorship Ladder

Progression in OpenNest:

```
Open Community
    ↓
Domain Pods
    ↓
Contributor
    ↓
Pod Lead / Mentor
    ↓
Core Council
```

Promotion is **output-based**, not title-based.
